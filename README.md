# 🏥 LABEASE - Pathology ERP Backend

A scalable and secure backend system for managing pathology laboratories, client subscriptions, appointments, billing, and reports.

Built with **Node.js, Express, MongoDB, and JWT Authentication**.

---

## 🚀 Features

### 🔐 Authentication & Authorization
- JWT-based authentication
- Role-based access control (Admin, Client, Patient)
- Secure middleware protection

### 🧾 Subscription Management
- Create subscription
- Update subscription
- Delete subscription
- Get subscription by email
- Check subscription active status

### 📅 Appointment Management
- Book appointment
- Update report values
- Generate reports

### 💳 Billing System
- Generate bill
- Save bill
- Generate bill PDF

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Nodemon
- dotenv
